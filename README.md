- `docs/` — 28 fichiers de documentation projet (specs, archi, sécurité, API, modules, etc.)

## Modules

- **Monitor** — Track costs in real-time, smart alerts, cost forecast, model recommendations
- **Replay** — Step-by-step visual timeline of every agent session for debugging
- **Protect** — Configurable guardrails, PII redaction, budget caps, kill switch
