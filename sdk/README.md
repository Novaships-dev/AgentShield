# agentshield

The official Python SDK for [AgentShield](https://agentshield.one) — the complete observability suite for AI agents.

## Installation

```bash
pip install agentshield
```

## Quick Start

```python
import agentshield

# Configure once at startup
agentshield.configure(api_key="ags_live_your_key_here")

# Track a model call
# (full tracking API available in v0.2.0)
print(agentshield.__version__)  # 0.1.0
```
